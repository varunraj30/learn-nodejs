# Guide

- Create a account on Discord
- Create a server
- Enable Developer mode for server
- Go to Discord Developer portal
- Create a new App
- Go to Bot - Create Bot (Admin Privileges)
- Go to Oauth2 - URL Generator - bot,admin - copy url & paste
- Add bot to your Server

[Discord Docs](https://discord.js.org/docs/packages/discord.js/14.26.2)
[Discord Developer Portal](https://discord.com/developers/applications)
